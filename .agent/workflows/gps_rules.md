# Regras para Geração de GPS

1. **Nunca incluir juros e multa** nos valores das GPS HTML. Usar apenas o "Valor Contribuição" (valor base). Outro sistema é responsável por calcular juros e multa.

2. **Sempre separar competências** em meses individuais, mesmo quando o período no CNIS aparece como faixa contígua (ex: "01/11/2018 a 31/12/2018" deve gerar duas linhas: 11/2018 e 12/2018).
